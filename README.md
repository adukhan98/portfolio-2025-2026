## ‚òÅÔ∏è Deployment (Netlify)

This project is optimized for **Netlify**. 

1.  Connect your GitHub repository to Netlify.
2.  It will auto-detect the Next.js runtime (thanks to `@netlify/plugin-nextjs`).
3.  **Build Command**: `npm run build`
4.  **Publish Directory**: `.next`

### A Note on the Contact Form üì¨
The contact form currently uses a concise simulation (it simulates a network request). To make it functional without a backend:
-   **Option A**: Enable **Netlify Forms**. Add `data-netlify="true"` to the `<form>` tag in `src/components/ContactForm.tsx`.
-   **Option B**: Use [EmailJS](https://www.emailjs.com/) to send emails directly from the client.
